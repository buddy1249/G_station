Fuel Accounting Service (Flask + Peewee)

Простое веб-приложение на Python для учета выдачи топлива собственному и коммерческому транспорту. Система позволяет вести ежедневную отчетность, отслеживать заправки по сменам и формировать итоговые данные за месяц.
Основные функции:

    Учет собственного транспорта: регистрация заправок с привязкой к водителю, путевому листу и номеру колонки.
    Учет сторонних организаций: контроль отпуска топлива коммерческим клиентам и физлицам с фиксацией суммы оплаты.
    Ежедневные отчеты: автоматическое формирование таблиц с объемами топлива (суточные, накопительные и итоговые показатели).
    Фильтрация по датам: просмотр истории заправок и статистики за конкретные смены.
    

Технологический стек:

    Backend: Python (Flask)

Database: Peewee ORM (SQLite/PostgreSQL)
Templates: Jinja2 (HTML)

Как запустить:

    Установите зависимости: pip install flask peewee
    Настройте модели в файле model.py.
    Запустите приложение: python main.py
    Откройте в браузере: http://127.0.0.1:5000
